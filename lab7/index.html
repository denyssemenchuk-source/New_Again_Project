Завдання з JS

Завдання 1: Пріоритет скриптів

Відповідь: Першим виконається big.js, оскільки браузер виконує стандартні синхронні скрипти строго в тому порядку, в якому вони розташовані в HTML-документі.

Завдання 2: Логіка та змінні

  • Чому дорівнює x: x = 5. Спочатку виконуються дужки (a *= 2, a стає 4), потім додавання (1 + 4 = 5).
  • Чи з'явиться alert: Так, з'явиться. У JavaScript будь-який непорожній рядок (навіть "0") при приведенні до логічного типу стає true.
  • Змінні: Результат коду зі змінною admin ("Василь") виводиться у консоль браузера.
